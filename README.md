## ğŸš€ Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
   git clone https://github.com/FaisalMubeen2001/my-portfolio.git
   cd my-portfolio
```

2. Install dependencies
```bash
   npm install
```

3. Start the development server
```bash
   npm start
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production
```bash
npm run build
```

The production-ready files will be in the `dist` folder.

## ğŸ”§ Customization

To customize this portfolio for your own use:

1. **Personal Information**: Update name, title, and bio in `src/portfolio.jsx`
2. **Projects**: Modify the `projects` array with your own projects
3. **Skills**: Update the `skills` object with your technical skills
4. **Experience**: Change the `experiences` array with your work history
5. **Resume**: Replace the PDF in the `public` folder
6. **Images**: Add your profile image to `public/images/`
7. **Colors**: Search and replace color codes in the CSS section
